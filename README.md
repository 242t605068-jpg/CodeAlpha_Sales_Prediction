# Sales Prediction using Python

## Description
This project predicts future sales using machine learning techniques in Python. The analysis focuses on understanding how advertising spending influences sales performance. Data preprocessing, visualization, and regression modeling are used to forecast sales and provide actionable business insights.

## Objectives
- Clean and preprocess sales data.
- Analyze relationships between advertising spend and sales.
- Build a regression model for sales prediction.
- Evaluate model performance.
- Generate insights for marketing strategies.

## Tools and Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Conclusion
The project demonstrates how advertising investments can influence sales outcomes. Predictive modeling helps businesses optimize marketing budgets and improve decision-making.
